# Forkify 是一個串接真實 API 仿購物車清單練習


### 技術方面

使用的技術包含許多 ES6 語法，當中使用了： 
- async / await 異步處理
- 箭頭函式
- 正則表達式
- 解構賦值
- export 及 import 概念
- axios 串接技術


### 工具方面

工具方面則是使用了以下這些技術：
- npm 套件管理工具
- babel 轉譯器(ES6 -> ES5)
- webpack 打包工具
- fraction.js 函式庫
- uniqid.js 自動產生id函式庫

### 開發測試

開發時的測試練習則是使用 docker 指令解決了跨域代理的問題。
